Push Button Metrics

Push Button Metrics is a Power BI workspace with reports for Resilient Incidents and Kill Chain.

It provides a consolidate picture of the total and movement of detections on phishing, web reputation, anti-malware, serious intrusion, serious compromise and incidents.


(click on the image to open in a new tab)


Data Sources

CTU Kill Chain metrics relies on a variety of feeds processed from the SDS Data Lake, reading CEF files generated by FluentD.

AMDC

  • ironport
  • zscaler
  • falcon
  • epo

EMDC

  • ironport
  • zscaler
  • windows_defender
  • epo

Data Refresh

Kill Chain report is refreshed daily 04:00 AM UTC.
